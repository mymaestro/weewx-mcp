# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-12

### Added
- Initial release of weewx-mcp
- `get_current_conditions` tool for retrieving current weather readings
- `query_temperature_range` tool for temperature statistics over date ranges
- `query_rainfall` tool for rainfall analysis
- `find_wind_events` tool for wind speed analysis
- Stdio transport support (local/SSH)
- SSE transport support (HTTP)
- Comprehensive documentation and setup instructions
